# eslint-config-node-strict

Requires Eslint >= 4

## Installation

`npm install eslint-config-node-strict`

## Usage

```
// .eslintrc
{
  "extends": "eslint-config-node-strict",
  // or
  // "extends": [
  //   "my-favourite-config"
  //   "eslint-config-node-strict"
  // ],
  "rules": {
    // omited
  }
}
```
